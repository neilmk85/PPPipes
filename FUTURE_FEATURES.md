## Mobile — Production Entry Sheet

### Shift Selector (A / B / C)
The production entry bottom sheet has shift support disabled (hidden in UI). The backend field `shiftName` accepts "A", "B", or "C". When this is enabled, show a row of three toggle buttons (A / B / C) below the date/search bar so the user can tag which work shift the entry belongs to.
